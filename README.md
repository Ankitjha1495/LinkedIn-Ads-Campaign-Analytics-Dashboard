# LinkedIn Ads Campaign Analytics Dashboard | Power BI
Project Overview

This Power BI dashboard provides a comprehensive analysis of LinkedIn advertising campaign performance. The dashboard enables marketing teams and business stakeholders to monitor campaign effectiveness, optimize advertising spend, track lead generation, and evaluate overall return on investment (ROI).

The report transforms raw campaign data into actionable insights through interactive visualizations, KPI tracking, and dynamic filtering capabilities.

Business Objective

The primary objective of this project is to analyze LinkedIn Ads campaign performance and provide a centralized reporting solution for measuring marketing effectiveness across different campaigns, industries, regions, and devices.
